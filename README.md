## Gradle

### Pulling the dependency 

Put this in the top of the build.gradle at the root of your project:

```groovy
buildscript {
  repositories {
    mavenCentral()
    mavenLocal()
  }

  dependencies {
    classpath 'com.darylteo.gradle:javassist-plugin:0.2.0'
  }
}
````
